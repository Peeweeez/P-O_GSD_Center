# P-O GSD Center

A modern, minimalist **Getting Stuff Done (GSD)** productivity hub designed for professionals who need to juggle multiple projects, tasks, deadlines, and ideas—all in one beautifully designed interface.

**P-O GSD Center** combines the power of task management, strategic prioritization (Eisenhower Matrix), deadline tracking, and knowledge capture—bringing clarity to chaos.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or installation required

### Quick Start

1. **Open the Application**
   - Simply open `P-O GSD Center Modern Minimalist.html` in your web browser
   - Or use a local server (recommended for best performance):
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (with http-server)
     npx http-server
     ```

2. **Create Your First Project**
   - Click the **"＋ Project"** button in the top-right
   - Enter a name (e.g., "Q4 2024 Goals")
   - Choose a color for easy visual identification
   - Set the status (Active, On Hold, Complete)

3. **Add Tasks**
   - Navigate to **Tasks** in the sidebar
   - Click **"+ Task"** button
   - Fill in title, description, priority, due date, etc.
   - Subtasks and comments can be added in the task detail view

4. **Start Organizing**
   - Use the **Matrix** tab to organize tasks by urgency/importance
   - Add **Deadlines** to track critical dates
   - Capture **Ideas** and **Notes** for context

## 💾 Data Storage & Export

### How Data is Stored
- **LocalStorage** – All data is stored in your browser's local storage
- **No Cloud Sync** – Your data never leaves your device
- **No Account Required** – Complete privacy and anonymity

### Backing Up Your Data

#### Export
1. Click **"⬇ Export"** in the top toolbar
2. A JSON file will download automatically
3. Backup name includes the current date: `po-gsd-backup-2024-12-16.json`

#### Import
1. Click **"⬆ Import"** in the top toolbar
2. Select a previously-exported JSON file
3. Your data will be restored (existing data will be overwritten)

**💡 Tip:** Export your data regularly to ensure you have backups!

## 📋 Project Concepts

### Projects
A container for all related tasks, deadlines, notes, ideas, links, and snippets. Think of them as:
- Work projects
- Personal goals
- Side hustles
- Life areas (Health, Family, Finance, etc.)

### Quadrants (Eisenhower Matrix)
```
              IMPORTANT    |    NOT IMPORTANT
URGENT        Do First     |    Delegate
NOT URGENT    Schedule     |    Eliminate
```

- **Do First** – Crisis, deadlines, emergencies (Red)
- **Schedule** – Planning, prevention, improvement (Yellow)
- **Delegate** – Interruptions, activities (Blue)
- **Eliminate** – Time wasters, unnecessary (Gray)

### Task Status
- **Todo** – Not started yet
- **In-Progress** – Currently working on it
- **Done** – Completed

### Task Priority
- **Critical** – Must do immediately
- **High** – Important
- **Medium** – Standard importance
- **Low** – Can wait

## ❓ FAQ

**Q: Is my data safe?**
A: Yes! Your data never leaves your device. It's stored entirely in your browser's local storage.

**Q: Can I use this on multiple devices?**
A: Yes, but you'll need to manually export/import data between devices since there's no cloud sync.

**Q: What if I clear my browser cache?**
A: Your data will be deleted. Always maintain backups by exporting regularly.

**Q: Can I share projects with others?**
A: Currently no, but you can export and share JSON files. Future versions may include collaboration.

**Q: Does this work offline?**
A: Yes! The entire application runs offline with no internet required.

**Q: How much data can I store?**
A: LocalStorage typically allows 5-10MB per domain, which is plenty for thousands of tasks.
